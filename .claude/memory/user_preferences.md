- Magyar nyelven kommunikál; technikai termek és kódazonosítók maradhatnak angolul
- Rövid, tömör válaszokat preferál — nincs szükség hosszú magyarázatokra
- Kérdez, mielőtt döntést hoz ("Mit gondolsz?") → elvárja a véleményt/javaslatot
- Szívesen fogad proaktív javaslatokat (pl. potenciális problémák felhívása)
- manage.sh-t preferálja a Makefile felett (rugalmasság)
- Nem szereti a duplikációt (pl. két hivatkozás ugyanarra az oldalra)
- Minimalista függőségek elve — új library hozzáadása előtt indoklás kell
